Git Workflow

  • git checkout main => switches to the main branch
  • git status => checks what branch we are currently on
  • git checkout -b branch-name => creates a new branch and switches to it
  • Use Git flow steps to add, commit, and push our work to our feature branch
  • git add -A => stages all content in the directory to the working branch
  • git commit -m "message for commit" => commits the changes
  • git pull origin main => refreshes the main branch
  • git push origin feature/feature-name => pushes the changes
